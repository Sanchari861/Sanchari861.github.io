Project 1: Global Scholar

This is the first project I completed at the end of Unit 1, and it was to test our skills in coding lines of basic Python. I was paired with a classmate, and we divided the work evenly. I coded the "Climate Action" and "Life Below Water" sections, which included all the different types of code to create the image, as well as the emblem in the center. Concepts I learned in this unit included creating shapes and understanding various properties, such as fill, border, opacity, and roundness, among others. I also learned about the basics of coding at CMU, including how inspectors and autocheckers work.

Concepts Learned

  • Python basics including colors and shapes
  • Python canvas basics, including size, inspector, other mechanics
  • Design mechanics and shape properties
